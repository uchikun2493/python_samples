# python_samples

### Overview

pythonで書いたサンプルコードを残していく

### Requirements

- pyhton 3.7
- opencv-python 4.0

### Install

```
$ git clone https://github.com/uchikun2493/python_samples.git
```

### /scripts

#### /scripts/cv_video.py

- リアルタイム性は考慮していない
